v1.8 Barbarendörfer Berichte-Auswertung

🏴 Barbarian Village Former

Wird jetzt vom Versammlungsplatz aus gestartet (nicht mehr von der Berichte-Übersicht) und lädt deine Angriffsberichte gegen Barbarendörfer im Hintergrund automatisch über alle Seiten der Berichte-Liste nach. Aus den enthaltenen Spähdaten ermittelt es Mauer- und Gebäudelevel der Barbarendörfer und berechnet passende Truppenkombinationen aus einer gewählten Dorf-Gruppe: erst die Mauer mit Rammen/Äxten brechen, danach das gewählte Gebäude in mehreren Schritten mit Katapulten herunterstufen (bis zu einem Mindestlevel, in wählbaren Level-Schritten pro Befehl). Läuft jetzt auch in der mobilen Tribal-Wars-App.

Das Ergebnis ist eine Liste fertiger Workbench-Befehle (Base64-codierte Truppenzahlen) zum Kopieren – kompatibel mit dem TW Format-Konverter dieser Seite. Es wird nichts automatisch abgeschickt.

Installation (Desktop)

Ziehe diesen Button in deine Lesezeichenleiste. Öffne danach im Spiel den Versammlungsplatz und klicke auf das Lesezeichen, um das Script zu starten.

🏴 Barbarian Village Former starten

Installation (Handy: iPhone/Android)

Auf dem Handy lässt sich das Bookmarklet nicht per Drag & Drop in die Lesezeichen-/Favoritenleiste ziehen. Stattdessen legst du zuerst ein ganz normales Lesezeichen an und ersetzt dessen Adresse danach durch den folgenden Code: 

javascript:$.getScript('https://marlongsimo.github.io/DS-Scripts/scripts/barbarianVillageFormer.js');

iPhone (Safari):

  1. Diese Seite (oder eine beliebige andere) über das Teilen-Symbol als Lesezeichen speichern.
  2. Unten auf das Bücher-Symbol tippen → „Lesezeichen“ → „Bearbeiten“ → das eben erstellte Lesezeichen antippen.
  3. Das Adressfeld leeren und den oben kopierten Code einfügen. Als Namen z.B. „Barbarian Village Former“ eintragen und speichern.
  4. Damit es schnell erreichbar ist: das Lesezeichen in den Ordner „Favoriten“ verschieben – es erscheint dann in der Schnellzugriffsleiste beim Öffnen eines neuen Tabs.
  5. Im Spiel am Versammlungsplatz einen neuen Tab öffnen und das Lesezeichen aus den Favoriten antippen.

Android (Chrome):

  1. Über das Menü (⋮) → „Lesezeichen“ → „Hinzufügen“ ein neues Lesezeichen anlegen.
  2. Das Lesezeichen antippen und halten → „Bearbeiten“, die URL löschen und durch den oben kopierten Code ersetzen, Namen anpassen und speichern.
  3. Damit es in der Schnellleiste erscheint: in den Chrome-Einstellungen unter „Lesezeichen“ die Lesezeichenleiste aktivieren, oder das Lesezeichen zum Startbildschirm hinzufügen.
  4. Im Spiel am Versammlungsplatz das Lesezeichen aus der Leiste antippen.

Nutzung auf eigene Verantwortung. Das Script sendet nichts automatisch ab, sondern erzeugt nur eine kopierbare Befehlsliste. Zusätzlich lädt es beim Start eine externe Bibliothek (twSDK.js) von twscripts.dev nach – es besteht also eine Laufzeitabhängigkeit von einer Drittanbieter-Domain. Bitte vorab die Regeln deines Servers zu Skripten/Bookmarklets prüfen. Das Script liest zum Auswerten immer die allgemeine Angriffsberichte-Liste, niemals einen deiner selbst angelegten Berichte-Ordner (z.B. „Hochladen“, „Archiv“) – im Panel wird nach jedem Lauf angezeigt, welche Berichte-Liste tatsächlich verwendet wurde.

DS-Scripts ist ein inoffizielles Fan-Projekt für „Die Stämme“ und steht in keiner Verbindung zu InnoGames.
