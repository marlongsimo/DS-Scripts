🎯 Mass Snipe

Berechnet für mehrere Snipe-Ziele gleichzeitig, wann welche eigene Dorf-Gruppe welchen Einheitentyp losschicken muss, damit der Angriff exakt zur gewünschten Ankunftszeit landet. Für jede Kombination aus Ziel-Koordinate, gewünschter Ankunftszeit, Sigel-Bonus (prozentualer Geschwindigkeits-Malus/-Bonus) und Mindest-Truppenanzahl werden alle passenden eigenen Dörfer aus der aktuell gewählten Gruppe durchsucht: Distanz, gewählte Einheitengeschwindigkeit und Sigel-Wert ergeben die Abschickzeit; anschließend wird geprüft, ob im jeweiligen Dorf tatsächlich genug von der gewählten Einheit vorhanden ist. Nur wirklich durchführbare Kombinationen (liegen in der Zukunft und haben genug Truppen) landen in der Ergebnistabelle.

Zusätzlich gibt es einen Massenimport: Text aus einem Forenthread (z.B. eine Liste angekündigter Adelszüge) einfügen, das Script erkennt Koordinaten und Ankunftszeiten automatisch und legt daraus passende Snipe-Zeilen an. Das Ergebnis lässt sich als BB-Code exportieren, und ein Live-Countdown zeigt an, wie viel Zeit bis zum nächsten Abschicken bleibt.

Installation (Desktop)

Ziehe diesen Button in deine Lesezeichenleiste. Öffne danach das Spiel und klicke auf das Lesezeichen, um das Script zu starten.

Installation (Handy: iPhone/Android)

Auf dem Handy lässt sich das Bookmarklet nicht per Drag & Drop in die Lesezeichen-/Favoritenleiste ziehen. Stattdessen legst du zuerst ein ganz normales Lesezeichen an und ersetzt dessen Adresse danach durch den folgenden Code: 

javascript:$.getScript('https://marlongsimo.github.io/DS-Scripts/scripts/massSnipe.js');

iPhone (Safari):

  1. Diese Seite (oder eine beliebige andere) über das Teilen-Symbol als Lesezeichen speichern.
  2. Unten auf das Bücher-Symbol tippen → „Lesezeichen“ → „Bearbeiten“ → das eben erstellte Lesezeichen antippen.
  3. Das Adressfeld leeren und den oben kopierten Code einfügen. Als Namen z.B. „Mass Snipe“ eintragen und speichern.
  4. Damit es schnell erreichbar ist: das Lesezeichen in den Ordner „Favoriten“ verschieben – es erscheint dann in der Schnellzugriffsleiste beim Öffnen eines neuen Tabs.
  5. Im Spiel einen neuen Tab öffnen und das Lesezeichen aus den Favoriten antippen.

Android (Chrome):

  1. Über das Menü (⋮) → „Lesezeichen“ → „Hinzufügen“ ein neues Lesezeichen anlegen.
  2. Das Lesezeichen antippen und halten → „Bearbeiten“, die URL löschen und durch den oben kopierten Code ersetzen, Namen anpassen und speichern.
  3. Damit es in der Schnellleiste erscheint: in den Chrome-Einstellungen unter „Lesezeichen“ die Lesezeichenleiste aktivieren, oder das Lesezeichen zum Startbildschirm hinzufügen.
  4. Im Spiel das Lesezeichen aus der Leiste antippen.
