v1.6.9 (Mod) Barbarendörfer Bauernhof-Assistent

🏴 Clear Barbarian Walls

Liest den Bauernhof-Assistenten (alle konfigurierten Seiten) aus und listet daraus alle Barbarendörfer mit bekannter oder unbekannter Mauer auf – sortiert nach Entfernung, mit direktem Link zum zugehörigen Spähbericht. Statt einzelner Angriffslinks pro Dorf erzeugt das Script ein einziges Bookmarklet: im Versammlungsplatz gezogen/eingefügt und wiederholt angeklickt, trägt es automatisch das nächstgelegene, noch nicht abgearbeitete Barbarendorf mit der zum Mauerlevel passenden Truppenzahl ein (Äxte, Rammen, ein Späher) – den Angriff musst du dort selbst bestätigen, es wird nichts automatisch abgeschickt. Der Fortschritt bleibt im Browser gespeichert (auch nach einem Reload). Die Screen-Erkennung nutzt den vom Spiel gesetzten Wert statt nur die Browser-Adressleiste, damit das Script auch in der mobilen Tribal-Wars-App zuverlässig erkennt, dass es im Bauernhof-Assistenten läuft. Das Overlay wird außerdem in der App in den richtigen Container eingefügt (dort gibt es kein #contentContainer wie im Desktop-Browser, sondern #mobileContent).

Diese modifizierte Version passt die Truppenzahlen an: Äxte folgen der Formel 25 × Mauerlevel (Level 1 = 25, Level 2 = 50, usw.), und liegt kein Spähbericht vor (Mauer unbekannt), werden die Werte für Mauerlevel 1 verwendet (25 Äxte, 4 Rammen).

Installation (Desktop)

Ziehe diesen Button in deine Lesezeichenleiste. Öffne danach im Spiel den Bauernhof-Assistenten – das Script scannt automatisch und zeigt am Ende der Übersicht ein neu generiertes Bookmarklet an, das du dann im Versammlungsplatz wiederholt anklickst.

Installation (Handy: iPhone/Android)

Auf dem Handy lässt sich das Bookmarklet nicht per Drag & Drop in die Lesezeichen-/Favoritenleiste ziehen. Stattdessen legst du zuerst ein ganz normales Lesezeichen an und ersetzt dessen Adresse danach durch den folgenden Code: 

javascript:$.getScript('https://marlongsimo.github.io/DS-Scripts/scripts/clearBarbarianWalls.js');

iPhone (Safari):

  1. Diese Seite (oder eine beliebige andere) über das Teilen-Symbol als Lesezeichen speichern.
  2. Unten auf das Bücher-Symbol tippen → „Lesezeichen“ → „Bearbeiten“ → das eben erstellte Lesezeichen antippen.
  3. Das Adressfeld leeren und den oben kopierten Code einfügen. Als Namen z.B. „Clear Barbarian Walls“ eintragen und speichern.
  4. Damit es schnell erreichbar ist: das Lesezeichen in den Ordner „Favoriten“ verschieben – es erscheint dann in der Schnellzugriffsleiste beim Öffnen eines neuen Tabs.
  5. Im Spiel im Bauernhof-Assistenten einen neuen Tab öffnen und das Lesezeichen aus den Favoriten antippen.

Android (Chrome):

  1. Über das Menü (⋮) → „Lesezeichen“ → „Hinzufügen“ ein neues Lesezeichen anlegen.
  2. Das Lesezeichen antippen und halten → „Bearbeiten“, die URL löschen und durch den oben kopierten Code ersetzen, Namen anpassen und speichern.
  3. Damit es in der Schnellleiste erscheint: in den Chrome-Einstellungen unter „Lesezeichen“ die Lesezeichenleiste aktivieren, oder das Lesezeichen zum Startbildschirm hinzufügen.
  4. Im Spiel im Bauernhof-Assistenten das Lesezeichen aus der Leiste antippen.
